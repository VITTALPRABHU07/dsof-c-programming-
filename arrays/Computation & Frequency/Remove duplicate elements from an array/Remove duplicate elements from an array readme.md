Program: Remove Duplicate Elements from an Array
 Aim
To remove duplicate elements from a given array.

 Description
This program accepts a set of elements into an array and removes repeated (duplicate) values.
Each element is compared with the remaining elements in the array.
When a duplicate is found, it is removed by shifting the remaining elements to the left.
The final array contains only unique elements.

 Algorithm
Start
Read the number of elements
Read all array elements
Compare each element with the remaining elements
If a duplicate is found:
Remove the duplicate element
Shift remaining elements to the left
Reduce the array size
Display the array without duplicate elements
Stop

 Sample Input
Number of elements: 6  
Array elements: 1 2 3 2 4 1

 Sample Output
Array after removing duplicates:  
1 2 3 4

 Conclusion
The program successfully removes duplicate elements from the array and displays only unique values.
